## How to get the images

These images were generated via AI image generation during development.
See the chat session or ask Claude to regenerate them with:

- Player: `Pixel art sprite sheet reference for Kael, top-down alchemist character, 16x16, idle+walk 4 directions`
- Enemies: `Pixel art enemy reference sheet — Ashburn Shambler, Iron Hulk, Silver Specter, Gold Golem, Uranium Wraith, Platinum Sentinel`
- Bosses: `Pixel art boss reference — PEROXIS (sodium peroxide crystal), AURIUM (golden serpent), URANOX (uranium stone giant), 48x48`
- Tileset: `Pixel art tileset — 6 biome rows: Surface Plains, Underground Mines, Crystal Caverns, Sky Islands, Ocean Floor, Magma Layer`
- UI/Buildings: `Pixel art UI reference — alchemist workshop, market stall, well, Synthesizer machine, research desk, HUD health bar`

See `../SPRITE_SPEC.md` for full animation specs and frame counts.
